Un e-mail vous sera envoyé. Les fichiers sont des fichiers CSV directement importables depuis Gephi. Pour ce faire:

  • - créz un nouveau projet
  • - allez dans l'onglet " Data laboratory"
  • - cliquez sur "ImportSpreadSheet"
  • - selectionnez le premier fichier cvs : temp_node_XX.csv ( attention , il faut commencer par importer le fichier de noeuds avant le fichier de liens )
  • - Selectionnez
  • - separator : comma
  • - as table : nodes table
  • - Charset : UTF-8-
  • - Enfin cliquez sur "next" puis "finish"
  • - Faire de meme pour le fichier de lien : temp_edge.csv ( attention a bien selectionner as table : edges table)
