Carto Bessif UTC


Bienvene sur notre outil de crawl pour la cartographie des brevets. Projet dans le cadre d'une UV au sein de l'UTC

Ce site vous permettra de remplire une base de données avec les brevets correspondant à vos mot clefs de recherche. Nous vous proposons pour le moment deux services : - télécharger le contenu de la base en fichier csv importable sous géphi. - remplir la base en effectuant une nouvelle recherche par mot clefs. N'oubliez pas que la base de données est commune ;Merci de ne pas lancez pas de recherche sur des mots clefs déjà scrawlés.
